# Forgetting curve

The decline of memory retention over time. Hermann Ebbinghaus showed in 1885
that, without reinforcement, recall drops sharply at first and then levels off —
an exponential decay.

The practical response is [[spaced-repetition]]: schedule each review near the
point where recall would otherwise fail, so the curve is reset with minimal
effort.
